# C1 Milk Admin

A comprehensive milk delivery management system built with React, Google Apps Script, and Cloudflare Pages.

## Architecture

- **Frontend**: React 19 + Vite 8 (PWA-enabled)
- **Backend**: Google Apps Script (serverless)
- **Edge Proxy**: Cloudflare Pages Functions
- **Database**: Google Sheets

## Features

- Customer management with delivery schedules
- Daily delivery logging and tracking
- Automated monthly billing with payment tracking
- Bulk milk import management
- Pause period management
- Credit notes and adjustments
- WhatsApp integration for bill sharing

## Setup

### Prerequisites

- Node.js 18+
- Google account with Apps Script access
- Cloudflare account (for deployment)
